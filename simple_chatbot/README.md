# Simple Chatbot

A basic conversational chatbot implementation in Python.

## Description

This is a simple chatbot that responds to user input using keyword matching. It's a great starting point for learning about natural language processing and conversational AI basics.

## Features

- Interactive command-line interface
- Basic keyword-based responses
- Simple conversation flow

## How to Run

```bash
python simple_chatbot.py
```

Type your messages and press Enter. Type 'quit', 'exit', or 'bye' to end the conversation.

## Customization

You can enhance this chatbot by:
- Adding more response patterns
- Implementing sentiment analysis
- Using NLP libraries like NLTK or spaCy
- Adding a memory system to remember previous conversations
- Creating different personality traits
